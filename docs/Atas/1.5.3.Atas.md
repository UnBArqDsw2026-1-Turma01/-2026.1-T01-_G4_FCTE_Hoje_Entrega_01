# 1.5.3. Atas de Reunião

## Introdução

Uma Ata de Reunião é um documento formal que registra tudo o que foi discutido e decidido durante um encontro. Ela serve como registro oficial das decisões tomadas, das responsabilidades atribuídas, dos prazos definidos e das informações relevantes compartilhadas entre os participantes.

## Objetivo

Os propósitos centrais da adoção dessa metodologia no projeto foram:
- Registrar de forma clara e organizada os assuntos tratados.
- Servir como referência futura para os participantes e para aqueles que não puderam comparecer.
- Garantir que decisões e responsabilidades fiquem formalmente documentadas.
- Auxiliar no acompanhamento das tarefas e dos prazos estabelecidos.

## Metodologia

As atas das reuniões foram documentadas seguindo o seguinte padrão:

- Identificação da reunião, conforme o modelo definido na **Tabela 1**.
- Listagem dos participantes presentes, padronizada conforme **Tabela 2**.
- Registro dos pontos discutidos.
- Registro das decisões tomadas.
- Definição dos próximos passos.
